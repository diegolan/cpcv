# Sistema de Vendas

Sistema criado para simplificar o processo de pedidos no PDV para varejo.

## Pré-requisitos

Precisa ter o Java 8 instalado na máquina.

## Uso

Basta clicar duas vezes no executável acai.jar ou executar via linha de comando:

```cmd
java -jar acai.jar
```
